# VLSI Design Internship – Task 2
## Introduction to Verilog HDL and RTL Design of Basic Combinational Circuits

## 📌 Objective :
This repository contains the implementation of **Task 2** of the MainCrafts VLSI Design Internship. 
The objective of this task is to learn the fundamentals of **Verilog HDL**, understand **RTL (Register Transfer Level) Design**, and implement basic combinational logic circuits along with their testbenches.

---

##  Concepts Covered :
- Verilog Hardware Description Language (HDL)
- RTL Design
- Verilog Modules
- Continuous Assignment (`assign`)
- Testbench Writing
- Simulation and Waveform Analysis

---

##  Implemented Modules :
### Logic Gates -
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate

### Combinational Circuits-
- Half Adder
- Full Adder

##  Simulation
Each module was verified using a dedicated testbench. The output waveforms were analyzed to ensure correct functionality according to the respective truth tables.
